# 什么是线性最小二乘法

# 前言（废话）

线性回归因为比较简单，可以直接推导出解析解，而且许多非线性的问题也可以转化为线性问题来解决，所以得到了广泛的应用。

甚至许多人认为最小二乘法指的就是线性回归，其实并不是，最小二乘法就是一种思想，它可以拟合任意函数，线性回归只是其中一个比较简单而且也很常用的函数，所以讲最小二乘法基本都会以它为例。

但网上很多教程都只是讲了它的公式以及运用，却没有一步一步推导的过程，即便略有涉及，也是三言两语，草草概之。

所以今天这里会用一个简单的例子来简单阐述一下最小二乘法的概念。

余之拙见，函望指正。

# 最小二乘法的一个简单例子

这个有点难讲，这里讲一个最简单的点斜式方程。
就是

y=ax+by = ax + by=ax+b

不知道小学现在有没有教，但想必大家都是即为熟悉。

最小二乘法的最简单的一个例子就是对散点图进行线性拟合，如图所示。

其中，我们可以看见，这条红线位于这些红点之间，我们也知道在这个例子里，最小二乘法的目的就是为了拟合出一条能够最贴近各点均值的一条最优解直线。
那这条直线是如何实现的呢？

# 最小二乘法的简单式子

\tag{1}

y^=ax^+b\hat{y}=a\hat{x}+by^​=ax^+b

这个函数就是我们所假设的拟合好之后的最优解直线
其中的 y^\hat{y}y^​ 是我们所输入的数据的 y 的值，而 x^\hat{x}x^ 就是我们所输入的数学的 x 的值。
对，就是那些红点的 x 值与 y 值。
而 y^\hat{y}y^​ 和 x^\hat{x}x^ 是我们所已知的

Δ=∑i=1ny^i−yi(2)\tag{2} \Delta = \sum_{i=1}^n \hat{y}_{i} - y_{i} Δ=i=1∑n​y^​i​−yi​(2)

如上式 δ\deltaδ 就是每一个点在 x 值相同情况下，y^\hat{y}y^​ 与 yyy 的插值，即是点到假设已存在的最优解拟合直线的距离。

但有同学已经发现，平时我们所看见的例子之中，公式 2 里的 ∑i=1ny^i−yi\sum_{i=1}^n \hat{y}_{i} - y_{i}∑i=1n​y^​i​−yi​ 这一部分是带有平方的。是这样没错那么我们将式 2 左右皆开一个平方。
则

δ=(Δ)2=(∑i=1ny^i−yi)2(3)\tag{3}\delta = (\Delta)^2 = (\sum_{i=1}^n \hat{y}_{i} - y_{i})^2 δ=(Δ)2=(i=1∑n​y^​i​−yi​)2(3)

如此以来这个式子应该就和大家平日所看见的一样了。
然后
将（1）代入（3）中

δ=(Δ)2=∑i=1n(y^i−ax^−b)(y^i−ax^−b)\delta = (\Delta)^2 = \sum_{i=1}^n(\hat{y}_{i} - a\hat{x} - b)(\hat{y}_{i} - a\hat{x} - b) δ=(Δ)2=i=1∑n​(y^​i​−ax^−b)(y^​i​−ax^−b)

即

δ=∑i=1n(y^i)2+x^2a2+....+b2;(4)\tag{4}\delta = \sum_{i=1}^n (\hat{y}_{i})^2 + \hat{x}^2a^2 + .... + b^2; δ=i=1∑n​(y^​i​)2+x^2a2+....+b2;(4)

省略掉了一些中间项，但不用在意，因为重点在于这两部分 x^2a2\boxed{\hat{x}^2 a^2}x^2a2​ 和 b2\boxed{b^2}b2​
我们已经知道的是其中的 a 与 b 是未知量，其它的 y^\hat{y}y^​ 和 x^\hat{x}x^ 都是已知量。
而我们所假设出来的式 1 中，也是如此。也就是说，我们只要求出了 aaa 与 bbb 的值，就可以知道我们所拟合出来的这条直线的方程了。
那么 aaa 和 bbb 怎么求呢？

# 求 a 与 b 的值

注意我们的式（4）
接下来我们先放下上面的论证，回想我们初中高中就有教过的一个简单的方程式

y=ax2+bx+cy = ax^2 + bx +c y=ax2+bx+c

在这个式子里，aaa bbb ccc 是常数，而 x 与 y 一个是自变量一个是因变量。
根据初高中的所学的二次函数的图像与性质，我们可以知道，当 aaa > 0 时，该式函数图形开口向上。
具体如图所示：

而我们还知道有一个数学方法叫作微分求极值。
我们使 yyy 对 xxx 求导，可以得其斜率，而当其求导所得的斜率为 0 时，我们可以知道这是 yyy 的极值，又因为这个函数图形开口向上，所以我们可以知道该极值为极小值，又因为是二次函数，所以同时也是它的最小值。

同理，我们回到我们的式 4.
其中 aaa 和 bbb 是变量，aaa 的系数是 x^2\hat{x}^2x^2 , 可知一个数的平方必达于 0，所以该函数图像必定开口向上。
那么目前我们所有的就是 一个有两个变量的二元二次方程式。
亦可以理解为两个二次函数 y=x^2a2y= \hat{x}^2a^2y=x^2a2 和 y=b2y=b^2y=b2 的叠加，
我们分别对 aaa 和 bbb 进行偏微分。
即

\left\{ \begin{aligned} \tag{5}\frac \partial y \ \partial a = (\hat{y}_{i} - a\hat{x} - b)\hat{x} \\ \tag{6}\frac \partial y \ \partial b = (\hat{y}_{i} - a\hat{x} - b) \end{aligned} \right. \begin{cases} \tag{5}\frac \partial y \ \partial a = (\hat{y}_{i} - a\hat{x} - b)\hat{x} \\ \tag{6}\frac \partial y \ \partial b = (\hat{y}_{i} - a\hat{x} - b) \end{cases}

求其极小值，即求最小值，即令 式 (5) 与式（6）皆为 0。
如此我们便能得到一个二元一次方程组。

(y^i−ax^−b)x^=0(y^i−ax^−b)=0 (\hat{y}_{i} - a\hat{x} - b)\hat{x} = 0 \\ (\hat{y}_{i} - a\hat{x} - b) = 0 (y^​i​−ax^−b)x^=0(y^​i​−ax^−b)=0

\left\{ \begin{aligned} \tag{7} (\hat{y}_{i} - a\hat{x} - b)\hat{x} = 0 \\ \tag{8} (\hat{y}_{i} - a\hat{x} - b) = 0 \end{aligned} \right.

联立式（7）与（8）我们即可求得 aaa bbb 的值，从而得到最优线性回归直线。

# 最后小记

最小二乘法的英文是 least square method。
其中的 square 在我看来，即有平方的意思。
最小二乘法让我感觉最妙的地方就是这个平方，求其方差，这里可以使函数开口向上，求极值，也就是求 δ\deltaδ 的最小值， δ\deltaδ 就是所有点对那条最优解直线的距离的值的平方求和，当 δ\deltaδ 最小时，也就意味着这个这条直线达到最优解的位置了。
